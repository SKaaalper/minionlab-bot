## Installation

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/ahlulmukh/streamai-bot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd streamai-bot
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
